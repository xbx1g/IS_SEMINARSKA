# AutoServis - Informacijski sistem za avtomobilski servis

**Avtor:** Marko Koren (63240159)

## Opis projekta
Sistem AutoServis digitalizira delovanje avtomobilskih servisov z reševanjem ključnih problemov ročnega beleženja terminov in komunikacije s strankami.

## Funkcionalnosti
- Spletna rezervacija terminov
- Upravljanje z urnikom mehanikov
- Evidence servisnih nalogov
- Digitalno potrjevanje predračunov
- Zgodovina servisov po strankah

## Tehnologije
- ASP.NET Core MVC 8.0
- Entity Framework Core
- SQL Server
- Bootstrap 5
- ASP.NET Core Identity

## Namestitev
1. Kloniraj repozitorij
2. `dotnet restore`
3. `dotnet ef database update`
4. `dotnet run`
